# Cómo abrir el Dashboard

## Paso 1 — Abrí una terminal en esta carpeta
(clic derecho en la carpeta → "Abrir en terminal" o desde VS Code: Terminal → Nueva terminal)

## Paso 2 — Ejecutá este comando
```
n8n-manager dashboard
```

## Paso 3 — El navegador se abre solo en:
**http://localhost:3456**

---

## Si el navegador no se abre automáticamente
Copiá y pegá esta URL en tu navegador:
```
http://localhost:3456
```

---

## Para agregar un nuevo cliente desde el dashboard
1. Abrí el dashboard con el comando de arriba
2. Hacé clic en **"+ Agregar cliente"** (arriba a la derecha)
3. Completá: Nombre, URL del n8n, y API Key
4. Clic en **"Conectar y guardar"**
